Retro 10

Overview

Retro is a Forth implementation which runs on a virtual stack machine called Ngaro. There are several implementations of Ngaro, allowing a high degree of portability.

This implementation of the VM is written in Java and can be used as either an applet or an application. It's not as polished as the other implementations, but does work.

The integrated image is the 10.0 release candidate image from December 2008.

To begin, click "Evaluate", then wait until the "ok" prompt appears.

You need a Java-enabled browser to view this applet.
